## 📖 專案簡介 | About

本應用程式專為輔仁大學學生設計，可快速查詢114學年度下學期各時段的空教室資訊，方便自習、小組討論或臨時借用空間。

This Android app is designed for Fu Jen Catholic University students to quickly find available classrooms during the 114th academic year's second semester — ideal for self-study, group discussions, or finding a quiet space on campus.

---

## ✨ 功能介紹 | Features

| 功能 | Feature |
|------|---------|
| 🔍 依時段查詢空教室 | Search available classrooms by time slot |
| 🏢 依教學大樓篩選 | Filter by building |
| 📅 查看每日空室時間表 | View daily availability schedule |
| 📌 快速查看教室容納人數 | Check classroom capacity |
